# Zylo - Premium Nigerian Soups Delivery

Beautiful website for authentic Nigerian soup delivery (Egusi, Afang, Fisherman Soup, etc.)

## Features
- Fully responsive design
- Dark & Light mode
- WhatsApp ordering
- Menu with prices and descriptions
- Gallery & Testimonials

## Tech Stack
- React + Vite
- Tailwind CSS
- Lucide React Icons

## How to Run Locally
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
